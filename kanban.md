# Kanban

This project uses GitHub Issues as its Kanban board.

- **Board**: https://github.com/Ezio0/agent-engineering-workflow/issues
- **Workflow**: Ezio creates issues → Ezio Zero implements → PR review → close
- **Priority labels**: P0 (blocking), P1 (important), P2 (nice-to-have)
- **Type labels**: enhancement, bug, documentation
