## Mandatory gates

Each item must be ✅. No exceptions.

- [ ] **WHO** is one specific person, not a demographic. Their age, role, location, and the moment your project enters their life are all named.
- [ ] **WHY** describes a pain that exists without your product. If your product disappeared tomorrow, this pain would still be felt.
- [ ] **WHY NOW** identifies a specific change (technology / data / regulation / behavior / incumbent failure) that did not exist 3 years ago.
- [ ] **UNDERLYING LOGIC** explains the mechanism, not the conclusion. If you remove the technical mechanism, the approach would not work — and you can explain why.
- [ ] **ANTI-POSITIONING** lists at least 3 things this project is NOT, each with a reason.

## Quality gates

Strongly recommended (not strict, but skipping them usually means the memo isn't ready).

- [ ] The entire memo fits on **one printed page** (≤ ~500 words). If longer, you haven't decided yet.
- [ ] The WHO is someone **you've either personally observed or could recognize in a coffee shop**. Not a market segment.
- [ ] You can defend each anti-positioning in **one sentence each**. "Not X because we don't solve Y" — and Y is a real adjacent problem.
- [ ] The WHY NOW is something that would **not** have been true if you'd started this project 3 years earlier.
- [ ] You have **not** used any of the following vague differentiators in positioning:
  - "AI-powered" / "intelligent" / "smart"
  - "Personalized" (without specifying the data source and mechanism)
  - "Seamless" / "frictionless"
  - "Revolutionary" / "next-generation"

## Self-check questions

Answer each in 1-2 sentences. If you can't, the memo needs another pass.

1. **If I told a friend about this project at a dinner party, could I explain it in 30 seconds?** What would I say?
2. **What is the cheapest possible test of this positioning?** (A landing page? A 5-person user interview? A concierge MVP?) Have you done it?
3. **What would have to be true about the world for this project to FAIL even if I built it well?** (This is the inverse of "why now" and is a stronger test.)
