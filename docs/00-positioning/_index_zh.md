## 目的

在写任何代码之前回答**为谁 / 为什么 / 底层逻辑 / 不是什么**。

## 计划 sections

1. 目标用户 / 客户分层
2. 问题陈述
3. 为什么是现在（市场时机 / 紧迫性）
4. 底层逻辑 / 核心 insight（为什么 work）
5. 价值主张（一句话）
6. 反定位（不是什么）
7. 成功标准（定性）
